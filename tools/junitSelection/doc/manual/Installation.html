Installations Instructions

    Once you have installed Σοφία on your computer, you are able to install Junit Test Selection. In this version, this software requires the version 3.8.2 of JUnit which can be downloaded here.

    The current version of Junit Test Selection can be downloaded here: SIR Tools

  1. Choose where to extract the JUnit Test Selection system. On extraction, a directory called "junitSelection" will be created in this location.
  2. Unzip the package using gunzip: gunzip junit-selection.tar.gz
  3. Extract the resulting archive: tar -xvf junit-selection.tar
    All the files should be extracted into subdirectories in the "junitSelection" directory. This directory is the root directory for the entire system.
  4. Add the location of JUnit Test Selection to your Java classpath. Suppose you extracted JUnit Test Selection into a directory named "tools" in your home directory. Then you would execute the following command (in csh/tcsh): setenv CLASSPATH "${CLASSPATH}":${HOME}/tools
  5. Add the libraries of Junit 3.8.2 and Σοφία to your classpath:
    setenv CLASSPATH "${CLASSPATH}":${HOME}/tools/junitSelection/lib/junit.jar:${HOME}/tools/sofya/lib/sofya.jar
